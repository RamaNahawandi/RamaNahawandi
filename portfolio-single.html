“Wireless glove for interpreting users gestures into a spoken language”

Project Details

  • Software Development Stack:
    Java and JSON for the Desktop application.

  • Hardware Development Stack:
    ESP32, Arduino, C.

  • Communication Model:
    TCP client server.

For the past years, it has been significantly challenging when it comes to interacting with non-speakers (deaf or non-vocal individuals). It appears that, it always required a human sign language translator to make any normal communication possible between the previous two. My final year project is titled “Wireless glove for interpreting user’s gestures into a spoken language”. The project explores the possibility of making the translation process from Sign Language to English autonomous by providing disabled people a wearable, portable, and a light-weight glove that will automatically captures their gestures and speak the corresponding word. The two key components of the system are: a glove; equipped with ESP32 microcontroller, flex sensors, and a three-axis accelerometer and gyroscope. That is to measure the flexion of the fingers, and to detect the orientation and rotational movement of the hand. The other component is the software application; to process the values sent from ESP32 and translate them to text and then to speech. The ESP32 microprocessor embedded in the glove analyzes the sensors readings to identify the gesture and outputs a vector of the gestures code. Then, it sends the gestures code to the software application wirelessly using the integrated Wi-Fi chip. The software is a Windows Java desktop application. The Java application parse each gesture code with the corresponding gesture meaning in English. Then it uses the FreeTTS library to speak the word. The software features: translating sign language gestures, adding new customized gestures whether they are letters, words, or phrases upon each user preference, translating texts into gestures by outputting the corresponding gesture move as an image. The project suggests an innovative affordable solution through the development of a prototype device to help practically bridge the communication gap between hearing and non-hearing people in a way that improves on the methods of pre-existing designs. In the future, the system can be developed to become an IoT device. Shrinking it to a standalone system connected to a cloud, without any interaction with a PC.
